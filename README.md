# Tetris — React · Next.js · Tailwind

A slick, fully playable Tetris implementation built using React, Next.js, and Tailwind CSS.

Originally built and repurposed for College Society (MLSC) use.

Play it live: [https://tetris-nine-amber.vercel.app/](https://tetris-nine-amber.vercel.app/)

---

## Features

* Classic Tetris mechanics with smooth, responsive controls
* Fixed 150-second timer for a fast, arcade-style session
* Fully responsive layout with mobile support
* Deployed live on Vercel with automatic GitHub-based updates
* Local high score tracking stored in the browser

---

## How To Play

1. Click "Start Game" to begin — the timer starts immediately and blocks begin falling.
2. Use the keyboard controls:

   * Left / Right Arrow: Move piece horizontally
   * Down Arrow: Soft drop
   * Up Arrow: Rotate piece
3. Clear horizontal lines to score and manage the stack height.
4. The game ends automatically when the 150-second timer expires.
